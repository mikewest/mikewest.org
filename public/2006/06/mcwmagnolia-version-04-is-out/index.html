`mcw_ma_gnolia` version 0.4 is out

Saturday, June 17, 2006

I’ve just updated mcw_ma_gnolia to 0.4, tweaking the plugin to work with Ma.gnolia’s new linkroll JavaScript. This, of course, is a good reason to start working on something that actually uses the Ma.gnolia API instead of parsing through a linkroll JavaScript file. I’ll be starting on that soon.

Download the new version and let me know if you run into any problems.

mcw_ma_gnolia is a TextPattern plugin that generates a customizable Ma.gnolia link roll for use on your website. If you’re curious, visit the project page for all the details and documentation.
