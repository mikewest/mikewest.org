Comments With Specificity

Monday, October 09, 2006

For a long time now, I’ve wanted to implement Flickr-style notes as a comment system for a peer-review system I put together a while back, but ran into so many cross-browser positioning issues that I more or less gave it up for dead. Jack Slocum might just have changed that.

I need to dig into the code a bit, but he’s basically reinvented the comment presentation system in a way that I’m really excited about. I’m not entirely convinced by this particular UI (clicking on a sidebar to post comments for a block seems a little obtuse), but I’m absolutely loving the idea of comments tied to individual blocks instead of global comments for an entire article. You’ll be hearing more about this from me later… :)

— Mike West
