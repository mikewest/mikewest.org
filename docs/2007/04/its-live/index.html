Tuesday, April 03, 2007

Yahoo! News has relaunched in the UK. Finally. This is the first project I’ve worked on at Yahoo!, and we’re proud to be “done.” Still some kinks to work out of the system, and some CSS bugs we’re (of course) only seeing now, but we’ll get those worked out.

In the near future, the news platform will be rolled out to other European markets, starting with Spain. For the moment, though, I’m going to go take a week-long nap. Yahoooo!

Update: And now I’m reloading the digg thread every 5 minutes, waiting for the first “YAH00 IZ L0OZ3RZ!” comments to start rolling in. I’m a maschoist, apparently. :)

— Mike West
